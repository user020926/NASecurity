# NASecurity

NASecurity 是一個用於管理 NAS (Network Attached Storage) 設備用戶的工具，支援用戶密碼變更、新增用戶和刪除用戶等功能。該程式透過 Synology NAS 的 API 操作，提供簡單的圖形化界面，無需額外安裝 Python 環境即可運行。

## 功能
- **管理員登入**: 使用 NAS IP、管理員帳號、密碼及雙重驗證碼（若需要）登入。
- **用戶管理**:
  - 更改現有用戶密碼。
  - 創建新用戶並生成隨機密碼。
  - 刪除指定用戶。
- **日誌記錄**: 操作結果自動保存為 Excel 檔案至桌面。
- **狀態顯示**: 在界面中即時顯示操作進度和結果。

## 系統需求
- **作業系統**: Windows 7 或更高版本。
- **網路**: 必須能連接到目標 NAS 設備。
- **權限**: 需要對桌面有寫入權限以保存日誌檔案。

## 安裝與運行
1. **下載程式**:
- 從提供的連結或儲存庫下載 `main.exe`。
2. **放置檔案**:
- 將 `main.exe` 放在您希望的目錄中（建議放在桌面或專用資料夾）。
3. **運行**:
- 雙擊 `main.exe` 啟動程式，無需額外安裝。

## 使用方法
1. **啟動程式**:
- 雙擊 `main.exe`，將打開 NASecurity 視窗。
2. **輸入資訊**:
- **NAS IP**: 輸入 NAS 設備的 IP 地址，例如 `10.57.78.62`。
- **管理員帳號**: 輸入有管理權限的帳號。
- **管理員密碼**: 輸入對應密碼。
- **雙重驗證碼**: 若啟用雙重驗證，輸入 6 位驗證碼，否則留空。
- **選擇文件**: 點擊「瀏覽」選擇包含用戶操作的 Excel 檔案。
3. **Excel 檔案格式**:
- 必須包含欄位: `帳號`, `作業需求`。
- 可選欄位: `工號`, `姓名`。
- `作業需求` 可填入: `密碼變更`, `新增用戶`, `刪除用戶`。
- 範例:
- 檔案必須為 `.xlsx` 格式。
4. **開始執行**:
- 點擊「開始執行」按鈕，程式將處理 Excel 中的每一行。
- 操作狀態會顯示在視窗下方的文字區域。
5. **查看結果**:
- 操作完成後，日誌檔案將自動保存至桌面：
- `NASecurity_log_YYYY-MM-DD.xlsx`: 記錄成功操作。
- `NASecurity_error_log_YYYY-MM-DD.xlsx`: 記錄失敗操作。

## 日誌檔案
- **成功日誌**: 包含時間、帳號、工號、姓名、執行結果及新密碼（若有）。
- **錯誤日誌**: 包含時間、帳號、工號、姓名及錯誤訊息。
- **儲存位置**: 預設保存至桌面。

## 注意事項
- **網路連線**: 確保 NAS 可透過指定 IP 存取，且防火牆未阻擋連接。
- **權限**: 管理員帳號需有足夠權限執行用戶管理操作。
- **Excel 檔案**: 請勿在程式運行時編輯 Excel 檔案，以免影響處理。
- **關閉程式**: 關閉視窗時，程式會自動保存日誌並登出 NAS 會話。

## 問題排查
- **程式無反應**: 檢查 NAS IP 是否正確，或確認網路連線。
- **日誌未生成**: 確認桌面有寫入權限，或嘗試以管理員身份運行程式。
- **錯誤訊息**: 查看視窗中的紅色文字，了解失敗原因。

## 聯絡支援
